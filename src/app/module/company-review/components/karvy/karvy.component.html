Karvy Trading Software (Karvy Trading Platforms)

Karvy Stock Broking offer multiple trading platforms based on customers investment style and needs.


Karvy Online Web

Karvy Online Web or Xpress platform is a web based trading platform which is also works great on low bandwidth. It provides secure and real time access to news, quotes, and trading. It provides convenience to trade anywhere on mobiles, laptop and tablet. Using this platform trader can place, modify, or cancel orders, view order book, trade book, Intraday book and back office reports and real time news.


KarvyOnline Mobile Trading App

KarvyOnline Mobile Trading App is a mobile platform which allows trader to execute trades on the move as well as access stock market news, currencies, commodities, quotes of NSE & BSE and many more updates instantly. KarvyOnline mobile app is available in Android Google Play and iPhone App Store for free.


Karvy Nest Trder

Karvy Nest (Next Generation Securities Trading System) Trader is desktop based front-end application that allows a trader to perform several trading functions. Karvy NEST Web is fast, convenient and secure mode for trading activities. It allows the investor to do all the things that are possible from a Trading Work Station (EXE) and more like view orders, trades and positions data, advance charting, view Market Picture and add-on tools to receive alerts, send internal messages etc.
